# nw-update-demo

``` bash
# 克隆项目
git clone https://github.com/zsj975516/nw-update-demo.git

# 安装服务端依赖
cd nw-update-demo/server && npm install

# 安装客户端依赖
cd nw-update-demo/client && npm install

# 打包客户端
cd nw-update-demo/client && npm run build

# 安装客户端
双击【nw-update-demo\client\build\nw-update-demo_win64_v1.0.0.exe】

# 修改版本号后打包客户端
修改【nw-update-demo\client\package.json】中的version修改为比之前安装版本高
执行打包客户端

# 启动服务端
双击【nw-update-demo/start server.bat】

# 启动之前安装的1.0.0
双击桌面快捷图标

# 检查更新

```
